A bit about me

How I got here

I started my journey with HTML and CSS, focusing on building clean and responsive interfaces. As I progressed, JavaScript introduced interactivity and helped me understand how modern web applications work, which strengthened my interest in building complete and functional products.

"I want to understand the full journey of a project — from writing the first line of code to deploying it live."

This curiosity led me to cloud computing, where I began exploring AWS to understand how real-world applications are hosted and scaled. Working with services like EC2, S3, and core networking concepts helped me connect development with deployment and infrastructure.

DevOps naturally became the next step. After deploying projects, I focused on improving workflows through Git, containerization with Docker, and basic CI/CD automation using GitHub Actions. I enjoy learning by building, experimenting, and continuously improving systems to make them more reliable and efficient.
